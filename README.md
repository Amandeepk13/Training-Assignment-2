# Training-Assignment-2

This assignment is a full stack app built using **Node.js, Express, MongoDB, Sessions, and Frontend JavaScript**.

## Features
- User authentication with **sessions**  
- Password stored securely with **bcrypt hashing**  
- Each user has their own classes.  
- Classes accessible via **unique UUID**
- Supports filtering rows by flag (true/false)
